# 梅花易數經典案例

## 一、觀梅占（邵康節原典）

### 背景
辰年十二月十七日申時，邵康節先生偶然觀賞梅花，見兩隻麻雀在枝頭爭鬥後雙雙墜地。

### 起卦過程
```
辰年 = 5
十二月 = 12
十七日 = 17
申時 = 9

上卦：(5 + 12 + 17) = 34 ÷ 8 餘 2 → 兌
下卦：(5 + 12 + 17 + 9) = 43 ÷ 8 餘 3 → 離
動爻：43 ÷ 6 餘 1 → 初爻動

本卦：澤火革
變卦：澤山咸（初爻變）
互卦：見乾巽
```

### 解卦
- **本卦分析**：兌金為體，離火克之，凶象
- **互卦分析**：巽木生離火，克體之氣更盛
- **動爻**：初爻動，事在近期
- **類象推斷**：
  - 兌為少女，將有女子受傷
  - 巽為股（大腿），傷在股部
  - 變卦艮土生兌金，不至凶危

### 斷語
> 「明晚當有女子折花，園丁不知而逐之，女子失驚墜地，遂傷其股。」

### 應驗
次日晚間，果然有鄰家少女前來折花，被園丁發覺追逐，驚慌墜地傷到大腿。

## 二、今人占買車案例

### 背景
某人問是否應該買 AWD 版本的電動車，還是省錢買 RWD 版本。

### 起卦
以當時時間起卦，得：
- 本卦：澤地萃（第45卦）
- 動爻：上六
- 變卦：天地否（第12卦）

### 解卦分析

#### 本卦【萃卦】
- 卦意：聚集、為將來而預備
- 兌上坤下：澤在地上，水聚之象
- 含義：需求會再次聚集，不只看現在

#### 動爻【上六】
- 位置：事在將來才顯現
- 爻義：「聚而未終，後悔可免」
- 解讀：當下看似用不到，將來會需要

#### 變卦【否卦】
- 天地不交，閉塞不通
- 警示：若只選省錢方案，未來會心不甘

### 斷語結論
```
✔ 建議：買 AWD

理由三點：
1. 萃卦：未來需求一定會聚集
2. 動在上爻：後期才顯現價值
3. 否卦警示：不買後必心裡不舒服

白話：此車表面是通勤，實際會拿去做超出通勤之事。
若現在不買 AWD，未來還是會後悔一次。
```

## 三、占天氣案例

### 情境
問明日天氣如何

### 起卦
數字起卦得：水天需卦

### 解卦
- 上坎下乾
- 坎為水、為雨
- 乾為天、為晴
- 卦名「需」有等待之意

### 斷語
> 明日上午晴朗，午後轉陰，傍晚可能有雨。
>
> 理由：乾天在下主早，坎水在上主晚。

## 四、占失物案例

### 情境
某人丟失手機，問能否找回

### 起卦
報數得卦：山水蒙，動爻在三

### 解卦
- 艮土為體，坎水為用
- 體克用，有望找回
- 艮為手，坎為暗處、水邊
- 蒙卦主啟蒙、被遮蔽
- 三爻動，變卦為山地剝

### 斷語
```
物可找回，但需仔細尋找。
位置：暗處、角落、水槽附近
時間：今明兩日內
注意：變卦剝，拖延恐難尋
```

## 五、占婚姻案例

### 情境
女方問與男友婚姻前景

### 起卦
時間起卦得：澤山咸，動在五爻

### 解卦
- 咸卦本意：男女感應，婚姻之卦
- 兌少女在上，艮少男在下
- 體用比和（皆土金）
- 五爻動，在君位
- 變卦：澤雷隨

### 斷語
```
🔮 大吉之卦

咸卦為婚姻正卦，男女相感
體用比和，門當戶對
變卦隨，隨遇而安，順其自然

建議：
- 順水推舟，自然發展
- 五爻動主貴人相助
- 今年秋季（兌金旺季）適合進一步
```

## 解卦技巧總結

### 1. 卦名直斷
六十四卦卦名本身就有含義，如：
- 咸 → 感應
- 損 → 減少
- 益 → 增加
- 困 → 困境

### 2. 體用生克
- 用生體 = 外來助力
- 體克用 = 我制約局面
- 用克體 = 受制於人
- 體生用 = 付出耗洩

### 3. 互卦看過程
互卦代表事情發展的中間階段

### 4. 變卦看結果
變卦代表最終走向

### 5. 卦氣論時間
- 體卦旺相之時，事易成
- 克體之卦當旺，要提防

### 6. 類象具體化
根據所問之事，取相應的八卦類象
